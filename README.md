# Node.js on Storm

Run Node.js on Apache Storm. 

## Run Example

```sh
docker-compose up -d
docker-compose run --rm nimbus /resources/example.js nimbus
open http://localhost:8080
```

Clean up:

```sh
docker-compose down -v
```
